# Segment-anything vue3

这是一个使用vue3+vite+ts+bun作为基础开发环境的segment-anything最小可运行demo，基于官方的react demo进行改造。(官方：<https://github.com/facebookresearch/segment-anything/tree/main/demo>)

## 项目启动

```shell
    bun i
    bun dev
```
